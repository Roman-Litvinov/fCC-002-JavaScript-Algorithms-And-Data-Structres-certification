// Augmented addition operator += can be used to concatenate
// a string onto the end of an existing string variable. This is
// useful when a very long string needs to be assigned and you
// want to break the operation over several lines, to make it more
// manageable.

// build a long string from 2 long parts:
var myStr = "This is the first sentence. ";
myStr += "This is the second sentence.";
